# DAYWORK
📌 Project Description

A real-time one-day job marketplace where people can instantly post small tasks and find available workers nearby.

The platform connects individuals who need immediate help (e.g. furniture assembly, moving, cleaning, small freelance tasks) with people who are ready to work the same day.

Users can:

Create and publish jobs in seconds
Browse and filter nearby opportunities
Apply for tasks and chat in real time
Track job status from creation to completion

The system is designed to feel like a living marketplace where opportunities appear and are fulfilled instantly.

⚙️ Key Features
Real-time job posting and discovery
Location-based matching
Instant messaging between users
Simple authentication (Google, Apple, guest access)
Unified user roles (everyone can both post and accept jobs)
Clean, minimal, mobile-first UI
Fully responsive design
Activity feed for job lifecycle updates
🧠 Backend Architecture
Powered by hidden n8n workflow automation
Event-driven system using webhooks
Handles:
Job lifecycle management
Worker matching
Notifications
Messaging triggers
Data validation
🎯 Goal

To build a fast, trustworthy, and intuitive platform for real-time human work exchange, focusing on simplicity and instant connections between people.
